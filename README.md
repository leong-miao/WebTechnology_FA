### Problem Statement
Currently, QIU cafeteria faces several issues during peak hours:
- **Long queues** and waiting time during lunch hours
- Students **missing classes** due to waiting for food
- Vendors struggling to **manage multiple orders** efficiently
- Orders being **misplaced or recorded incorrectly**
- Students **not informed** when their order is ready

### Solution
The QIU Online Ordering System provides:
- Online menu browsing
- Food pre-ordering with pickup time selection
- Real-time order status tracking
- Vendor dashboard for efficient order management
- Order history for students

## ✨ Features

### Student Features
| Feature | Description |
|---------|-------------|
| **Browse Menu** | View all available food items with descriptions, prices, and categories |
| **Filter by Category** | Filter menu by Malay, Chinese, Indian, Western, or Beverages |
| **Add to Cart** | Select items and quantities before ordering |
| **Place Order** | Submit pre-orders with preferred pickup time (15 mins to 2 hours ahead) |
| **Order History** | View all past orders by entering email address |
| **Track Status** | Check if order is Pending, Preparing, Ready, or Collected |

### Vendor/Admin Features
| Feature | Description |
|---------|-------------|
| **Dashboard** | View statistics including total orders, orders by status, today's orders, and today's revenue |
| **View All Orders** | See all incoming orders in a sortable table |
| **Update Status** | Change order status: Pending → Preparing → Ready → Collected |
| **Delete Orders** | Remove completed or cancelled orders |
| **Filter Orders** | Filter orders by status for easy management |
